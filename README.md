## 关于博客

该博客使用开源的 [jekyll] 构建于 [GitHub Pages] 之上。如需完整的 ```hold``` 住这个项目，需要掌握前端的基础知识以及 Jekyll 的基本的语法 [liquid] 等。

## 目录结构
```
blog
├── _includes
├── _layouts
├── _posts
└── assets
    ├── css
    ├── fonts
    ├── images
    └── js
```

可以详细的对照源码，注意放在 ```_posts``` 里面的文章的命名规则和形式。
